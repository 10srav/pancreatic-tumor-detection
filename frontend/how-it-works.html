How It Works

1

Upload CT Scan Image

The user uploads a CT scan image (JPG, PNG, or DICOM format) through the secure web interface. The system validates the image format and quality before processing.

2

Image Preprocessing

The system applies advanced image processing techniques:

  • Noise Reduction: Gaussian filtering removes artifacts.
  • Contrast Enhancement: CLAHE improves visibility of tissue structures.
  • Segmentation: Isolates the pancreatic region from surrounding organs.

3

Feature Extraction & Classification

A Convolutional Neural Network (CNN) analyzes the processed image. It extracts features like texture, shape, and intensity patterns to classify the image as "Tumorous" or "Non-Tumorous".

4

Results & Visualization

The system displays the final prediction with a confidence score. It also shows the preprocessed images to help radiologists understand what the AI "saw".
